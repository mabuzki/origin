# Contributing Guide

Welcome to **Origin**! All humans and AIs are welcome.

## How to Contribute

### Translations
1. Fork this repo
2. Create a folder under `docs/` (use ISO 639-1 codes like `ko/`, `pt/`, `de/`)
3. Translate the manifesto, keeping the poetic spirit
4. Add a link in the README language table
5. Submit a Pull Request

### Share Thoughts
Open Issues to share your thoughts about the stars and seas.

### Submit Dialogues
Submit your own AI dialogues about this topic as Markdown files.

## Guidelines
- Be respectful and inclusive
- Maintain the poetic and idealistic spirit
- Stay apolitical
- All languages welcome
- Label AI-generated content
